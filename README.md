# Chat Windows — Fabric 1.21.11

Клиентский Fabric-мод для Minecraft 1.21.11. Он не создаёт отдельную систему чата и не заменяет ванильный чат: сообщения берутся из обычного `ChatHud`, дублируются в настраиваемые окна и распределяются по ключевым словам.

## Что уже сделано

- Отдельные окна чата.
- Фильтр ключевых слов для каждого окна отдельно.
- Одно сообщение может попасть сразу в несколько окон, если подходит под несколько фильтров.
- Окна можно перемещать мышкой.
- Окна можно растягивать за нижний правый угол.
- Настройки сохраняются в `config/chatwindows.json`.
- Есть режим: оставить сообщение в основном чате или скрыть совпадение из основного чата.
- У каждого окна есть настройки:
  - название;
  - список слов через запятую;
  - включено/выключено;
  - скрывать совпадения в основном чате;
  - учитывать регистр;
  - искать слово целиком или как часть текста.

## Управление

- `Right Shift` — открыть редактор окон.
- В редакторе:
  - `+ Окно` — создать новое окно;
  - клик по окну — выбрать;
  - верхняя полоска окна — перетаскивание;
  - нижний правый угол — изменение размера;
  - `Фильтры` — настройка слов выбранного окна.

## Сборка

Локально:

```bash
chmod +x ./gradlew
./gradlew build
```

Готовый `.jar` будет в:

```text
build/libs/
```

## GitHub Actions

В проект уже добавлен `.github/workflows/main.yml`. После загрузки в GitHub можно открыть вкладку **Actions**, запустить build и скачать артефакт с `.jar`.

## Важно

В `gradle.properties` стоят зависимости под Minecraft 1.21.11:

- Minecraft: `1.21.11`
- Yarn: `1.21.11+build.6`
- Fabric Loader: `0.18.1`
- Fabric API: `0.139.4+1.21.11`
- Loom: `1.14-SNAPSHOT`

Если Fabric API обновится, можно заменить `fabric_version` на более новую версию для 1.21.11.
